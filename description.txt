流动性池管理：用户可向 AMM 流动性池中添加/移除多种代币。
代币交易/兑换：支持用户在流动性池中基于Stablecoin的数学模型实现代币兑换，包含滑点和手续费计算。
实时数据展示：显示流动性池中代币余额、价格变化曲线、交易记录等信息。
区块链交互：与智能合约交互，提交链上交易。
滑点控制：允许用户设置交易的最大滑点，避免因流动性不足或池价格波动造成的损失。
钱包集成支持：兼容 Web3 钱包（如 MetaMask）。
用户身份管理：通过 MetaMask 或其他 Web3 钱包进行身份验证，无需传统注册登录的方式。

前端：
框架： Next.js
UI 库：Tailwind CSS
钱包集成：Web3.js/ Ethers.js（整合 MetaMask）
图表展示：待定（用于显示池内资产变化、价格曲线等）

后端：（因为next.js集成了node.js所以也包含后端部分）
框架：Next.js 的 API 路由功能。
集成库：Ethers.js 或 Web3.js，与区块链交互。
缓存：Redis（是一种高性能的 开源分布式内存数据库 ，它可以用作 缓存、消息队列、持久化存储和实时数据处理 等功能），用于保存实时池状态，减少链上查询次数。
数据来源：
通过链上调用读取合约信息。
使用 The Graph 快速拉取 Curve 的历史交易或池状态。
